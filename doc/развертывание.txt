Скрипты для инициализации базы данных находятся в каталоге sql. В MySQL выполнить скрипт создания таблиц init.sql, а затем остальные скрипты для заполнения таблиц базовыми данными. Создать пользователя - прописать логин и пароль в таблице pr_users.

Скопировать содержимое каталога pr на HTTP-сервер. Скорректировать файл pr/template/connect.php, указав там логин, пароль, имя БД и ip сервера MySQL.
