Названия перемеменных, функций, таблиц, полей и всех других объектов должны быть на английском языке для возможности участие в проекте всех желающих независимо от их родного языка

SQL

При извлечении данных из нескольких таблиц объединять их оператором JOIN для большей наглядности
Ключевые слова пишутся заглавными буквами, имена таблиц и их полей - строчными
Названия таблиц пишутся в единственном числе - очевидно, что записей будет много
Поле таблицы с идентификатором должно называться id
Поле таблицы со ссылкой на идентификатор должно заканчиваться на "_id"
Названия внешних ключей должны начинаться с имени таблицы и включать имена полей, на которые распространяется ключ
При хранении дерева в таблице поле идентификатора родительской вершины должно называться id_hi. У корневых вершин значение id_hi is null (для выгрузки резервной копии)


PHP

При обращении к БД применяется PDO
